# mappa appalti — italia

Interactive map of Italian public procurement (aggiudicazioni) for 2024, by stazione appaltante. Source: [ANAC — BDNCP](https://dati.anticorruzione.it/opendata).

Single self-contained HTML artifact: open [`index.html`](index.html) in a modern browser. No server required.

- 23,954 stazioni appaltanti
- 1,036,692 aggiudicazioni
- 231,227 operatori distinti (dedup by codice fiscale)
- €824.5 mld totale aggiudicato (caveat: multi-year framework contracts are booked in full in the award year)

## Build

The HTML is generated by `build_html.py` from three small inputs in `raw/`:
- `agg_italia.json.gz` — gzipped per-(SA, CPV) aggregates (6.5 MB)
- `italy_regions.geojson` — 20 region polygons, ISTAT centroids (1.1 MB)
- `italia_places.json` — provincial capitals + comuni names with coordinates (0.4 MB)

```sh
python3 build_html.py
# → index.html
```

## Regenerating the data from scratch

The raw ANAC dumps used to compute the aggregates are not in the repo (multi-GB). To rebuild from source, download from [ANAC opendata](https://dati.anticorruzione.it/opendata):

- `aggiudicazioni` (full archive)
- `aggiudicatari` (full archive)
- `cig-2024` (12 monthly archives)
- `stazioni-appaltanti` (full)

Then run the processing pipeline (the steps are documented inline in this repo's git history).

ANAC's portal has a WAF that rejects non-browser User-Agents — pass `curl -A "Mozilla/5.0 ..."`.
